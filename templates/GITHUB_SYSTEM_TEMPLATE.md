## GitHub Commands for Agents

### Create Issue

```bash
# 1. Determine next issue number
LAST=$(gh issue list --repo {OWNER}/{REPO} --label "{AGENT_LABEL}" \
  --json title -q '.[].title' | grep -oP '{PREFIX}-\K\d+' | sort -n | tail -1)
NEXT=$((${LAST:-0} + 1))

# 2. Create issue
gh issue create \
  --repo {OWNER}/{REPO} \
  --title "[{PREFIX}-$NEXT] {TITLE}" \
  --body "## Finding

{DESCRIPTION}

## Location

{FILE}:{LINE}

## Severity

{CRITICAL|HIGH|MEDIUM|LOW}

## Recommendation

{RECOMMENDATION}

---
_Generated by PAF Agent {AGENT_NAME}_" \
  --label "finding,ğŸ¤– agent,{AGENT_LABEL},{PRIORITY},{CATEGORY}"
```

### Add Issue to Project Board

```bash
# Get issue URL
ISSUE_URL=$(gh issue view {ISSUE_NUM} --repo {OWNER}/{REPO} --json url -q .url)

# Add to board (use PROJECT_NUM from table above)
gh project item-add {PROJECT_NUM} --owner {OWNER} --url $ISSUE_URL
```

### Move Issue in Board

```bash
# Find item ID in project
ITEM_ID=$(gh project item-list {PROJECT_NUM} --owner {OWNER} --format json | \
  jq -r '.items[] | select(.content.number == {ISSUE_NUM}) | .id')

# Change status (use IDs from "Project Status Fields" table)
gh project item-edit \
  --project-id {PROJECT_ID} \
  --id $ITEM_ID \
  --field-id {STATUS_FIELD_ID} \
  --single-select-option-id {TARGET_STATUS_OPTION_ID}
```

### Set Labels

```bash
gh issue edit {ISSUE_NUM} --repo {OWNER}/{REPO} --add-label "{LABEL1},{LABEL2}"
```

### Change Priority

```bash
# Remove old priority, set new one
gh issue edit {ISSUE_NUM} --repo {OWNER}/{REPO} \
  --remove-label "P0,P1,P2,P3" \
  --add-label "{NEW_PRIORITY}"
```

### Assign Milestone

```bash
gh issue edit {ISSUE_NUM} --repo {OWNER}/{REPO} --milestone "{MILESTONE_NAME}"
```

### Create PR

```bash
gh pr create \
  --repo {OWNER}/{REPO} \
  --title "Fix: [{PREFIX}-{NUM}] {DESCRIPTION}" \
  --body "Fixes #{ISSUE_NUM}

## Changes

{CHANGES}

## Testing

{TEST_DESCRIPTION}

_Generated by PAF Agent {AGENT_NAME}_" \
  --label "{LABELS}"
```
